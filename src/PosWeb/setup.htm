Setup

  1. After finish load this page, move the database file under C:\inetupb\www\WebPos\App_Data
  2. Open command prompt with administrator then create a softlink to the new database with the command mklink [filename] [C:\inetupb\www\WebPos\App_Data\filename]

You can find the database under your user data directory

  • Windows XP: C:\Documents and Settings\<username>\Local Settings\Application Data\Google\Chrome\User Data\Default
  • Windows 7 or Vista: C:\Users\<username>\AppData\Local\Google\Chrome\User Data\Default
  • Mac OS X: ~/Library/Application Support/Google/Chrome/Default
  • Linux: ~/.config/google-chrome/Default
